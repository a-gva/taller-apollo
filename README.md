## Description

This is a dummy node graphql apollo server that returns mocked data and accepts migrations. Schema: user { id: String! (uuid autogenerated), name: String!, email: String! }. Use sqlite and drizzle for the database.

## How to run

```bash
pnpm install
pnpm run dev
```

## How to test

```bash
pnpm run test
```
